# Flutter Portfolio - Archived

This directory contains the original Flutter web portfolio code that has been archived.

## Archive Date
December 22, 2025

## Reason for Archiving
The portfolio has been migrated from Flutter to React.js + Tailwind CSS for better performance, easier maintenance, and modern web development practices.

## Contents
- `lib/` - Flutter source code
- `web/` - Flutter web configuration
- `test/` - Flutter tests
- `assets/` - Original Flutter assets
- `pubspec.yaml` - Flutter dependencies
- `pubspec.lock` - Flutter dependency lock file
- `.metadata` - Flutter project metadata

## Original Tech Stack
- **Framework**: Flutter 3.24.0
- **Language**: Dart
- **Renderer**: CanvasKit
- **Deployment**: Firebase Hosting

## Migration Details
The portfolio was successfully migrated to:
- **Framework**: React 18.x
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Deployment**: Firebase Hosting (same)

All content, features, and functionality have been preserved and enhanced in the new React version.

## Accessing the New Portfolio
The React portfolio is now in the root directory of this repository.

To run the new React portfolio:
```bash
cd ..
npm install
npm run dev
```

## Note
This Flutter code is kept for reference purposes only. The active portfolio is now the React version in the root directory.
